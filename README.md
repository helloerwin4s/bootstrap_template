# Bootstrap template

Bootstrap Template is simple thing to use bootstrap for your frontend environment.
Feature: 
- Bootstrap@^4.4.1
- JQuery@^1.9.1
- popper.js@^1.16.0


## Installation

```bash
npm install
```

## Serve development mode to your browser
```bash
npm run serve
```
Your browser automatically open http://localhost:9000/

## Build
```bash
npm run build
```
